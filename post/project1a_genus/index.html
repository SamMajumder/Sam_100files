Project #1A What are the most phenotypically divergent traits in wild sunflowers?

January 5, 2021

The code for this project can be found on my GitHub

Link to GitHub Repository

BACKGROUND:

Plants diversify across multiple trait axes and this is a hallmark of plant evolution. Some of these traits relate to the fitness of a species and also sheds light on the life history strategies. These traits are known as functional traits amongst ecologists and plant physiologists and tells us about how plants evolve and interact with their environment. In this present study I demonstrate the efficacy of a machine learning based approach in identifying the most divergent traits in the wild Helianthus genus. This genus is the common wild Sunflower.

KEY QUESTIONS:

A) What are the most divergent traits in wild sunflowers?

B) Can we use these potentially divergent traits to classify species accurately using a machine learning approach?

DATA:

The data used in this study was from a large greenhouse experiment conducted at the University of Georgia by Dr. Chase Mason (currently a Professor at the University of Central Florida). This data has been used to answer key plant evolutionary questions which you can read in the following scientific publications (links given below):

A) https://doi.org/10.5091/plecevo.2017.1321

B) doi:10.1111/evo.12768

C) doi: 10.1111/nph.13749

D) doi:10.1093/aob/mcx002

The analysis was conducted in the R programming environment.

METHOD:

Three parallel workflows were used to identify the most divergent trait axes and to model species classification using the most divergent traits. Each workflow differs in the type of feature selection method used. However, the machine learning classifiers used were the same in all three workflows. The classifiers being Random Forest and Gradient Boosting Machines. The feature selection methods used in separate workflows are Gini Impurity based feature selection (used in the first workflow), Recursive Feature elimination (used in the second workflow) and Boruta (used in the third workflow)

KEY FINDINGS:

An interactive dashboard showing the results from all three workflows and three most divergent trait axes can be found here https://sammajumder.github.io/Project_1A_Genus/Genus_report.html

The three workflows overlapped quite considerably with regards to what traits were deemed important at the genus level. Leaf trichome density, leaf circularity (shape of the leaf) and leaf area were traits that were identified as important traits by all three feature selection methods.
